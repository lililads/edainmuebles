# EDA aplicado a datos inmobiliarios
El presente análisis exploratorio de datos (EDA) tiene como objetivo comprender cómo influyen las características de un inmueble en la variación de su precio. Para ello, se estudiaron variables categóricas como Ciudad, Departamento, Estrato, Tipo de Inmueble y Detalle de Disponibilidad, combinadas con el valor del Precio. El EDA incluye análisis univariados, bivariados y multivariados con técnicas como histogramas y boxplots con escala logarítmica además de comparaciones entre múltiples categorías.
Este proyecto lo realicé como tarea para el curso virtual del SENA, Análisis Exploratorio de Datos en Python.
